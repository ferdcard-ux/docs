This quickstart walks you through everything you need to go from zero to a live search result. By the end, you'll have a Google account, a working API key, and a real response from the Search API sitting in your terminal. The whole process takes less than ten minutes.

## Prerequisites

Before you begin, make sure you have the following:

- A modern web browser to sign up and visit the API Console
- `curl` installed on your machine **or** Python 3.7+ for the code examples below
- A text editor or IDE to store your API key securely

<Warning>
  Never hard-code your API key directly into source code that you commit to version control. Use environment variables or a secrets manager instead. See the [Authentication](/authentication) guide for best practices.
</Warning>

## Steps

<Steps>
  <Step title="Create a Google account">
    Head to [google.com/signup](https://google.com/signup) and fill in your name, email address, and a strong password. If you already have a Google account, sign in at [google.com](https://google.com) — you're ready to move on.

    Once signed in, your account gives you access to personalized search history, saved preferences, and — most importantly for developers — the API Console.
  </Step>

  <Step title="Generate an API key">
    Navigate to the [Google API Console](https://console.google.com) and follow these steps:

    1. Click **Create Project** (or select an existing project from the dropdown).
    2. In the left sidebar, select **APIs & Services → Credentials**.
    3. Click **Create Credentials** and choose **API Key**.
    4. Copy the generated key and store it somewhere safe — you'll need it in the next step.

    <Note>
      You can restrict your API key to specific IP addresses or HTTP referrers from the **Edit API Key** screen. This is strongly recommended for production use.
    </Note>
  </Step>

  <Step title="Make your first search request">
    With your API key in hand, you're ready to query the Search API. Choose your preferred method below — both examples search for the query `"open source projects"` and return the top 5 results.

    Set your API key as an environment variable first:

    ```bash
    export GOOGLE_API_KEY="your_api_key_here"
    ```

    Now run the request:

    <CodeGroup>
      ```bash curl
      curl -G "https://api.google.com/search/v1" \
        --data-urlencode "q=open source projects" \
        --data-urlencode "num=5" \
        -H "Authorization: Bearer $GOOGLE_API_KEY"
      ```

      ```python Python
      import os
      import requests

      API_KEY = os.environ["GOOGLE_API_KEY"]
      BASE_URL = "https://api.google.com/search/v1"

      params = {
          "q": "open source projects",
          "num": 5,
      }
      headers = {
          "Authorization": f"Bearer {API_KEY}",
      }

      response = requests.get(BASE_URL, params=params, headers=headers)
      response.raise_for_status()

      data = response.json()
      for item in data["results"]:
          print(item["title"], "—", item["url"])
      ```
    </CodeGroup>
  </Step>

  <Step title="Interpret the response">
    A successful request returns an HTTP `200` status and a JSON body. Here's what a typical response looks like:

    ```json title="Example API Response"
    {
      "query": "open source projects",
      "total_results": 4820000000,
      "results": [
        {
          "rank": 1,
          "title": "GitHub: Where the world builds software",
          "url": "https://github.com",
          "snippet": "GitHub is where over 100 million developers shape the future of software, together.",
          "published_date": "2024-11-01"
        },
        {
          "rank": 2,
          "title": "Open Source Initiative",
          "url": "https://opensource.org",
          "snippet": "The Open Source Initiative is the steward of the Open Source Definition.",
          "published_date": "2024-10-15"
        }
      ],
      "next_page_token": "CAoQAA",
      "search_time_ms": 142
    }
    ```

    The key fields to know:

    | Field | Type | Description |
    |---|---|---|
    | `query` | string | The search term you submitted |
    | `total_results` | integer | Estimated number of matching pages in the index |
    | `results` | array | Ordered list of result objects |
    | `results[].rank` | integer | Position in the result set (1-indexed) |
    | `results[].title` | string | Page title as it appears in the index |
    | `results[].url` | string | Canonical URL of the result |
    | `results[].snippet` | string | Short excerpt from the page content |
    | `next_page_token` | string | Pass this value as `page_token` to fetch the next page |
    | `search_time_ms` | integer | Server-side query execution time in milliseconds |
  </Step>
</Steps>
