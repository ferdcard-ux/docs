## Get started in 3 steps

<Steps>
  <Step title="Create an account">
    Sign up at [google.com](https://google.com) to access personalized search, saved history, and API credentials.
  </Step>
  <Step title="Obtain an API key">
    Visit your [API Console](https://console.google.com) to generate a key for programmatic access.
  </Step>
  <Step title="Make your first request">
    Call the Search API with your key to retrieve results from the web.
  </Step>
</Steps>
